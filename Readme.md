## 系统介绍

- 本系统是商品管理系统，部署在Tomcat8.5.32 上，使用的Java开发工具包版本版本为JDK1.8 

- 数据库建立在本地，使用MySQL建立

## 相关功能

- 本系统实现了以下功能：
- 用户的登录与新用户的新增；
- 商品的购买；
- 商品的新增与删除；
- 商品详细信息的展示等

## 使用方法

- 项目部署后在浏览器地址栏输入http://localhost:8080/ 会自动载入登录界面，
- 用户输入用户名与密码登陆进入主页，尚未注册的用户可以根据提示信息先进行注册再登陆；
- 进入主页后，在搜索框里输入商品id进行商品搜索；
- 进入主页后，选择想要编辑的商品，点击进入编辑页面进行编辑（新增或者删除）；
